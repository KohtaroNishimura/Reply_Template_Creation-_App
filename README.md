# Reply Template Creation App

## 概要
- X（Twitter）のリプライ／引用リポストに対する返信文を、あらかじめ用意したテンプレートに沿って素早く整形するためのブラウザアプリです。
- 毎回同じようなプロンプトを手打ちする手間を省き、ChatGPTの任意のエージェントへ渡すテキストを一貫したフォーマットで出力します。
- ChatGPT API やサーバーは不要で、完成したテキストを ChatGPT 画面にそのまま貼り付けるだけで運用できます。

## 解決したい課題
- 返信作成を毎回 ChatGPT に依頼しているが、プロンプトや状況説明を手入力するのに時間がかかる。
- 引用リポスト／リプライなどパターンが複数あり、毎回状況説明を作り直すとミスも発生しやすい。

## 目指す体験
- テンプレートに沿ったフォームへ、元ポストの内容だけを貼り付ければ必要情報が揃う。
- リプライ／引用リポストの組み合わせを選択式で指定し、求められている返信の種類を明確化。
- 入力内容から ChatGPT に渡すプロンプトを自動整形し、ワンクリックでコピーできる。

## 入力イメージ
1. **元ポストの種類**: 「引用リポスト」または「リプライ」をラジオボタンで選択。
2. **相手からの反応**: 相手が自分のポストへ「リプライ」したのか「引用リポスト」したのかを選択。
3. **本文入力欄**: 元ポスト／相手ポスト／これから投稿したい文章をテキストエリアに貼り付ける。
4. **生成結果**: 上記の入力をもとに、ChatGPT に貼り付けるプロンプト＋整形済み文章を表示＆コピー。

### サンプルシナリオ
```
自分のポスト:
沖縄公演ついに来ましたね！しかも“音市場”とは…！格闘技で通い慣れた場所だけど、
ここでハイスタを観られるなんて胸アツすぎる。小学生無料は本当にありがたい🙏
これは行くしかない…！ #ハイスタ

相手の反応（引用リポスト）:
これは熱い！
絶対に行きたい💪🏾

出力したい文章（引用リポストで返信）:
これは熱い！
絶対に行きたい💪🏾
```
このような入力をまとめてプロンプト化し、ChatGPT へ渡すテキストとコピーしやすい整形文章を生成します。

## 使用技術（無料で完結）
- HTML（フォーム＋出力エリアのマークアップ）
- CSS（最低限のレイアウトとコピー完了表示などのスタイル）
- バニラ JavaScript（テンプレート管理、選択内容に応じた文面整形、クリップボードコピー）
- バックエンド・API 連携不要。静的ファイルのみで動作。

## ホスティング候補
GitHub Pages / Netlify / Vercel いずれかで静的ホスティング可能。ビルド工程は不要で、そのままアップロードするだけです。

## 今後の実装メモ
1. リプライ／引用リポストの組み合わせを管理するテンプレート定義オブジェクトを用意する。
2. 入力検証と「必須項目が足りない場合のアラート」を実装する。
3. プロンプトエリアに「ChatGPT に貼る説明」「整形後の文章」を分けて表示する。
4. コピー操作後にトーストやテキストでフィードバックを出す。

この README を起点に、最低限の HTML/CSS/JS だけで MVP を作り、まずは ChatGPT へ貼り付ける作業時間の短縮を目指してください。
